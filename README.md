# 🧲 Magnet Previewer 磁力链接预览

一个轻量级的 AstrBot 插件，直接调用 [whatslink.info](https://whatslink.info) API 实现磁力链接预览。

## ✨ 功能特性

- **自动触发：** 检测到磁力链接即自动解析。
- **信息展示：** 显示文件名、文件类型、总大小、文件数量和截图等关键信息。

## 🚀 使用方法

将本插件部署到 AstrBot 中后，无需任何操作，直接在聊天中发送磁力链接即可自动触发解析。

**示例：**

```
magnet:?xt=urn:btih:C14ED88B609F9C57A39067C51E80024AB7DCXXXX
```
## 📝 注意事项

- 本插件**硬编码**使用 `https://whatslink.info`，**大陆用户**若网络环境无法直接访问，需要自行解决网络代理问题。
